``dtmm`` is a simple-to-use light (electro-magnetic field) transmission and reflection calculation engine and visualizer. It can be used for calculation of transmission or reflection properties of layered homogeneous or inhomogeneous materials, such as liquid-crystal with homogeneous, periodic (diffraction gratings) or inhomogeneous director profile. *DTMM* stands for Diffractive Transfer Matrix Method and is an adapted Berreman 4x4 transfer matrix method. Details of the method are given in *... some future paper*.

Requisites
----------

* numba >= 0.45.0
* numpy >= 1.20
* matplotlib
* scipy


Optional (for faster fft):

* mkl_fft
* pyfftw

Installation
------------

Install with pip::

    $ pip install dtmm
    
or clone the latest development version and run::

    $ python setup.py install

Documentation
-------------

You can find the online manual at:

http://dtmm.readthedocs.io

Also, you may want to look at the examples/ directory for some examples
of use.
